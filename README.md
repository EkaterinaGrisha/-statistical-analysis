# Статистический анализ эффективности реабилитационных мероприятий

Данный проект представляет собой статистическое исследование динамики биометрических показателей (уровня гемоглобина) сотрудников трех независимых промышленных предприятий в процессе прохождения медико-социальной реабилитации.

## 📌 Описание исследования
Основная задача — верификация однородности терапевтического эффекта при применении единых медицинских протоколов на различных производственных площадках. В качестве целевой метрики используется **индивидуальный прирост гемоглобина ($\Delta Hb$)**.

## 🛠 Технологический стек
* **Python 3.12**
* **Библиотеки обработки данных:** Pandas, NumPy
* **Статистический аппарат:** SciPy (тесты Шапиро-Уилка, Левена), Statsmodels (ANOVA, Tukey HSD)
* **Визуализация:** Seaborn, Matplotlib

## 📊 Основные этапы анализа
1. **Препроцессинг:** Очистка данных, расчет дельта-показателей ($\Delta Hb = Hb_{After} - Hb_{Before}$).
2. **Проверка гипотез:** Подтверждение нормальности распределения и гомогенности дисперсий.
3. **Дисперсионный анализ:** Применение One-way ANOVA для выявления системного влияния фактора «Предприятие».
4. **Post-hoc анализ:** Попарные сравнения методом Тьюки для выделения статистически значимых различий между группами.
5. **Визуализация:** Построение Boxplots с буквенной нотацией значимости и тепловых карт p-значений.

## 📈 Ключевые выводы
* Установлено наличие статистически значимых различий в эффективности программ между предприятиями ($p < 0.05$).
* Идентифицирована площадка-лидер (Предприятие 1) с наиболее выраженным положительным откликом на терапию.
* Результаты обосновывают необходимость дифференцированного и персонализированного подхода к реабилитации взрослых в условиях промышленного производства.
